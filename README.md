## Audio Effects Framework


### Description

Audio effects library for use by plugin developers, designed for low-latency algorithms taking place frame-by-frame for buffered audio.


### Effects included in the library

```markdown
- Gain
- Pan
- Delay
- Biquad
- Distortion
- Compressor
- Expander
- Flanger
- Chorus
- Reverb
```
